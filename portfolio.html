Twitter Bot

In this project, I created a bot that would grab data from a database on my personal device, log onto twitter, tweet that data, then log out. This project was done using the Selenium library in Python. Check out the code on my github TwitterBot. The database on my personal device that the bot accesses was created using SQL. This process is automated, in the video I manually start up the program for video purposes but in actuality the program runs on its own. The way that I did this was by converting the .py file to a .exec file and setting a time on my calendar to open that .exec file twice a week.


Issues and Improvements

What would a project be without things to be made better (even if if never happens).

  1. One of the main issues I had was running this. I used a calandar to set the time for which this program would be run. While this did work, it also seemed to create a lot of undesired inefficiencies. One issue was that if my laptop was not on at the desired run-time, the program would not run until the laptop was open. Another issue is that if I was using my laptop at the run-time, then I would be interrupted by the launching of the executable file.

    A solution that I came up with for this lies within the raspberry pi I just got. If I can move all the code and SQL to the pi and run it using crontab then it would be always running and much more efficient.

  2. If this was to be something that a company uses, the database would need to be larger. Mine only contains 25 different possible tweets. The bot could also extract data from trending and stuff like that to be more representative of what is actually happening on twitter and not just randomly tweeting.

  3. It was actually difficult to use Selenium, I was only able to do this project because of my knowledge with CSS. If I tried showing this to someone who had no knowledge of CSS classes and things like that, it would take a lot of time for them to get it. I'm sure there might be a library already out there for logging onto Twitter via Python, but a future project for myself could be creating that library. I could create a library so if someone wants to, say, log on they input twitter. login("...") and that would select the login box and enter the username and password and then enter it all.


Data Structures
